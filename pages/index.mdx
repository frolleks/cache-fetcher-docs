# Introduction

## What is cache-fetcher?

It's a dead simple and opinionated data fetcher for JavaScript and React. It uses [axios](https://axios-http.com/) (specifically [redaxios](https://github.com/developit/redaxios)) under the hood.

## Install

**npm**:

```
npm i cache-fetcher
```

**yarn**:

```
yarn add cache-fetcher
```

**pnpm**:

```
pnpm add cache-fetcher
```
